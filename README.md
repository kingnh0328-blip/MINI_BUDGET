## 🔧 설치 방법

### 1. Python 설치
- Python 3.8 이상이 필요합니다.
- https://www.python.org/downloads/ 에서 다운로드

### 2. 필요한 패키지 설치
터미널(명령 프롬프트)에서 다음 명령어를 실행하세요:

```bash
pip install -r requirements.txt
```

개별 설치:
```bash
pip install streamlit pandas numpy plotly yfinance requests beautifulsoup4
```

---

## 🚀 실행 방법

터미널에서 다음 명령어를 실행하세요:

```bash
streamlit run budget_app.py
```

브라우저가 자동으로 열리며 `http://localhost:8501`에서 앱이 실행됩니다.

## ⭐ 주요 기능

### 1. 📊 대시보드
- **총 자산 현황**: 주식, 부동산 포함 전체 자산 확인
- **자산 구성 차트**: 파이 차트로 자산 비중 시각화
- **월별 지출 추이**: 바 차트로 지출 패턴 분석

### 2. 📈 주식 투자 관리
- **실시간 주가 조회**: Yahoo Finance API를 통한 실시간 데이터
- **수익률 계산**: 매수가 대비 현재가 수익률 자동 계산
- **가격 추이 그래프**: 시계열 차트로 주가 변동 확인
- **포트폴리오 관리**: 여러 주식 통합 관리

#### 지원하는 티커 형식:
- 🇺🇸 미국 주식: `AAPL` (애플), `GOOGL` (구글), `TSLA` (테슬라)
- 🇰🇷 한국 주식: `005930.KS` (삼성전자), `035720.KS` (카카오)

### 3. 🏠 부동산 관리
- **부동산 등록**: 아파트, 오피스텔, 빌라 등 다양한 유형 지원
- **시세 관리**: 취득가와 현재 시세 비교
- **평가손익 계산**: 자동으로 수익률 계산

### 4. 📅 일별 지출 기록
- **달력 형식 조회**: 월별로 일일 지출 확인
- **카테고리 분류**: 식비, 교통비, 쇼핑 등 자동 분류
- **지출 유형**: 고정지출/변동지출 구분
- **카테고리별 분석**: 파이 차트로 지출 비중 확인

### 5. ⚙️ 고정지출 관리
- **고정지출 등록**: 월세, 통신비, 보험료 등 정기 지출 관리
- **납부일 알림**: 각 항목별 납부일 설정
- **절약 제안**: AI 기반 지출 분석 및 절약 가능 항목 제시
- **고정/변동 비교**: 최근 3개월 지출 패턴 분석

## 📖 사용 가이드

### 1️⃣ 주식 추가하기

1. 좌측 메뉴에서 **"📈 주식 투자"** 선택
2. **"주식 추가"** 탭 클릭
3. 정보 입력:
   - 티커 심볼: 예) `AAPL` 또는 `005930.KS`
   - 수량: 보유 주식 수
   - 매수일: 구매한 날짜
   - 매수가: 주당 구매 가격
4. **"주식 추가"** 버튼 클릭

**팁**: 티커 심볼을 모르는 경우
- 미국 주식: Yahoo Finance에서 검색
- 한국 주식: 종목코드 + `.KS` (예: 삼성전자 = `005930.KS`)

### 2️⃣ 부동산 추가하기

1. 좌측 메뉴에서 **"🏠 부동산"** 선택
2. **"부동산 추가"** 탭 클릭
3. 정보 입력:
   - 부동산 이름: 예) "강남 아파트"
   - 유형: 아파트, 오피스텔, 빌라 등
   - 취득일: 구매 날짜
   - 취득가: 구매 가격
   - 현재 시세: 현재 시장 가격
   - 위치: 예) "서울시 강남구"
4. **"부동산 추가"** 버튼 클릭

### 3️⃣ 일별 지출 기록하기

1. 좌측 메뉴에서 **"📅 일별 지출"** 선택
2. **"지출 추가"** 탭 클릭
3. 정보 입력:
   - 날짜: 지출 발생일
   - 카테고리: 식비, 교통비 등
   - 금액: 지출 금액
   - 지출 유형: 변동지출 or 고정지출
   - 설명: 간단한 메모
4. **"지출 추가"** 버튼 클릭

**"달력 보기"** 탭에서 월별 지출 내역을 확인할 수 있습니다.

### 4️⃣ 고정지출 관리하기

1. 좌측 메뉴에서 **"⚙️ 고정지출 관리"** 선택
2. **"고정지출 설정"** 탭에서 정기 지출 등록:
   - 항목명: 예) "월세", "휴대폰 요금"
   - 금액: 월 납부 금액
   - 카테고리: 주거비, 통신비 등
   - 납부일: 매월 납부하는 날짜
3. **"지출 분석"** 탭에서 절약 가능 항목 확인

### 5️⃣ 대시보드에서 전체 현황 보기

1. 좌측 메뉴에서 **"📊 대시보드"** 선택
2. 주요 지표 확인:
   - 총 자산
   - 주식 자산
   - 부동산 자산
3. 자산 구성 파이 차트 확인
4. 월별 지출 추이 그래프 확인

## 💡 팁 & 주의사항

### 주식 데이터 조회
- **인터넷 연결 필수**: Yahoo Finance API를 사용하므로 인터넷 연결이 필요합니다.
- **티커 심볼 확인**: 정확한 티커 심볼을 입력해야 데이터가 조회됩니다.
- **한국 주식**: 종목코드 뒤에 반드시 `.KS`를 붙여야 합니다.
- **거래 시간외**: 장 마감 후에는 당일 종가가 표시됩니다.

### 데이터 저장
- 모든 데이터는 `budget_data.json` 파일에 자동 저장됩니다.
- 앱을 종료해도 데이터가 유지됩니다.
- 파일을 삭제하면 모든 데이터가 초기화됩니다.

### 절약 제안
- 최근 3개월의 데이터를 기반으로 분석합니다.
- 변동지출이 많은 카테고리를 우선적으로 표시합니다.
- 고정지출은 줄이기 어려우므로 변동지출에 집중하세요.

## 🔍 문제 해결

### Q: 주식 가격이 조회되지 않아요
**A**: 다음을 확인하세요:
1. 인터넷 연결 상태
2. 티커 심볼이 정확한지 (예: `AAPL`, `005930.KS`)
3. 해당 종목이 Yahoo Finance에 등록되어 있는지

### Q: 한국 주식 데이터가 안 나와요
**A**: 종목코드 뒤에 `.KS`를 꼭 붙이세요.
- 잘못된 예: `005930`
- 올바른 예: `005930.KS`

### Q: 그래프가 표시되지 않아요
**A**: 
1. 데이터가 충분히 입력되었는지 확인
2. 브라우저 캐시 삭제 후 새로고침
3. 다른 브라우저에서 시도

### Q: 앱이 느려요
**A**:
1. 주식 종목이 너무 많으면 조회 시간이 길어질 수 있습니다.
2. 불필요한 종목은 삭제하세요.
3. 브라우저를 재시작하세요.

## 📊 활용 예시

### 예시 1: 월급 관리
1. 고정지출 먼저 등록 (월세, 통신비, 보험료)
2. 일별로 변동지출 기록

3. 포트폴리오 탭에서 개별 종목 수익률 체크

3. 평가손익으로 투자 수익률 확인
